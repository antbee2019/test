CSS 有三种表现方式：内联式，嵌入式，外部式。
内联式：< 标签名 style="属性:值"></标签尾>。 style要写在开始标签里，可以用分号；添加多属性和值。
嵌入式：写在<head>里，格式如下。
  • <style type="text/css">
     标签名{
      属性1:值;
      属性2:值;
     }
    </style>
外部式：单独建立一个.css后缀的文件，然后在html中通过link标签调用，link写在head内。调用方式如下。

<link href="CSS文件名位置" type="text/css" rel="stylesheet" >

rel(relationship 联系 关系)属性指明文档与被链接文档的关系。 style(样式)sheet(表格)为样式表。

优先级关系：就近原则。 CSS语句执行为顺序执行，内联式写在标签里，所以优先级最高；
嵌入式和外部式，写在head中，谁越在下面，谁就会被最终执行。
如果要强制提升为最高级，在对应属性的值后，写入"!inmportant"即可(不包括分号，包括感叹号)。

这里只谈论了对所有标签的统一控制，若要想对使用相同的标签的不同对象进行控制，需要加为标签class或id，再另述。
